==================== CLASS METHOD ====================
* Class Hero
void display()         -> Menampilkan profil Hero
void getName()         -> Menampilkan nama Hero beserta salam
double getHealth()     -> Mengembalikan (return) jumlah Health Point
double getExperience() -> Mengembalikan (return) jumlah Experience Point
string setName()       -> Mengganti nama character Hero
void equipWeapon()     -> Menggunakan object weapon + atribut weapon
void equipArmor()      -> Menggunakan object armor + atribut armor
void battleInfo()      -> Menampilkan profil Hero + atribut weapon + atribut armor

* Class Weapon
void weaponInfo()      -> Menampilkan nama weapon + atribut weapon
string weaponName()    -> Mengembalikan (return) nama weapon
float getDamage()      -> Mengembalikan (return) jumlah Damage Point
float getMana()        -> Mengembalikan (return) jumlah Mana Point

* Class Armor
void armorInfo()       -> Menampilkan nama armor + atribut armor
string getName()       -> Mengembalikan (return) nama armor
float valueArmor()     -> Mengembalikan (return) jumlah Protection Point
float valueAbility()   -> Mengembalikan (return) jumlah Ability Point
